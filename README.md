# Print212 — site vitrine (HTML/CSS/JS)

Site vitrine **statique**, mobile‑first, ultra rapide et compatible **Netlify** pour l’imprimerie **Print212** (Fès, Maroc).

## Structure

- `index.html` : page unique (sections + SEO + formulaire Netlify)
- `css/style.css` : design clair/pro, responsive
- `js/main.js` : JS minimal (menu burger, message succès, année)
- `assets/logo.png` : emplacement du logo (à fournir)
- `sitemap.xml` / `robots.txt` : SEO

## Déploiement Netlify

- Déposer ces fichiers dans le dépôt.
- Sur Netlify : **Build command** = vide, **Publish directory** = `/` (racine du site).
- Le formulaire de contact fonctionne via Netlify Forms (attribut `data-netlify="true"`).
